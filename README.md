Projeto para controlar mão robótica com as próprias mãos utilizando a biblioteca mediapipe em Python.

Neste projeto vamos utilizar visão computacional e arduino.

Toda a programação é feita em Python, no o arduino vamos apenas instalar a biblioteca Pyfirmata Standard para comunicar com a programação Python.

Para esse projeto recomendo usar Python na versão 3.9
